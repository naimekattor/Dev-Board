Question-3: What is event delegation in the DOM?
Event deligation is a javascript technique of handling events where you delegate a parent element as a listner for all of its children.
Question-4: What is event bubbling in the DOM?
Event bubbling in the DOM is a process where the event starts at the target element and bubbles up the DOM tree.
